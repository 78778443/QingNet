## 背景

对目标检测项目之前一直是一知半解，为了加深我对深度学习目标检测的理解，手写了一个单目标检测项目


## 项目介绍

这个项目非常简单，只需要关注

data.py 数据迭代器
net.py 网络模型
predict.py 预测脚本
tools.py 辅助脚本,可以忽略
train.py 训练模型脚本

## 运行方式

把train.py运行起来即可，缺啥安装包用pip安装即可
